Esquema de Cores do CSS

O CSS (Cascading Style Sheets) oferece várias maneiras de especificar cores para elementos HTML. As cores podem ser definidas usando diferentes formatos, como nomes de cores, códigos hexadecimais, códigos RGB e códigos HSL.

Nome de Cor: red

Código Hexadecimal: #00ff00

Código RGB: rgb(0, 0, 255)
